== Question 1 ==

Open a file called "test.txt". Read the first line and print it out.

Answer:
-------
f = open("test.txt")
lines = f.readlines()
print lines[0]

== Question 2 ==

Open a file called "test.txt". Remove the content of the file and write "Hello" on the first line of the file.

Answer:
-------
f = open("test.txt", "wb")
f.write("Hello")
f.close()

== Question 4 ==

A file has content below (total of 4 lines in the file):

One
Two
Thre
Four

Write a progrem to randomly select a line, and print that line out.

Answer:
-------
import random
f = open("test.txt", "rb")
lines = f.readlines()
pick = random.randint(0, len(lines) - 1)
print lines[pick]

== Question 5 ==

A file has content below (total of 3 lines in the file):

joe:pass1
bob:pass2
tom:pass3

Read the file and store them into a dictionary called "users" so that the user's password can be printed based on their name.

Example: print users["joe"]

Answer:
-------
f = open("test.txt", "rb")
users = {}
for line in f.readlines()
	(k,v) = line.split(":")
	users[k] = v

print users["joe"]

== Question 6 ==

Write a Python code that will print the last line of the file. Assume the file name is "test.txt".

Answer:
-------
f = open("test.txt", "rb")
lines = f.readlines()
print lines[len(lines) - 1]
# OR
print lines[-1]
